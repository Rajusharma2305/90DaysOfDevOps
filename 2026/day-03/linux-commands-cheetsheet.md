
Process management:
>ps aux : to list all the running processes
>top : Live view of cpu and memory usage
>htop : better interactive version of htop
>kill : Stop a process by pid >kill -9 pid:To stop a process forcefully
>pkill : kill a process by its name
>jobs : shows background jobs
>bg/fg :run jobs in bg or fg



File system:
>pwd:Show current directory.
>ls -l:List files with details.
>cd directory_name:Change directory.
>mkdir folder:Create a new folder.
>rm -rf folderDelete files/folders.
>cp source destination:Copy files.
>mv old new:Move or rename files.
>find / -name file.txt:Search for files.
>cat file.txt:View file content.
>grep "word" file.txt : to search for a specific word



Networking troubleshooting:
>ping google.com : check internet connectivity
>ip a : shows ip address and network interfaces
>ss -tulnp : shows listening ports and services
>curl google.com : Test website response
>wget URL : To download fron internet
>traceroute google.com : shows network path to destination
