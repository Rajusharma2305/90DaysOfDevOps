Today’s goal is to practice basic file read/write using only fundamental commands.

You will create a small text file and practice:

Creating a file:touch myfile.txt
Writing text to a file:echo "Linux practice Day 06" > notes.txt
Appending new lines: echo "Learning file operations" >> notes.txt
echo "Using redirection operators" >> notes.txt
echo "Practicing cat command" >> notes.txt
echo "Reading files with head" >> notes.txt
echo "Reading files with tail" >> notes.txt
Reading the file back:head -n 2 notes.txt
tail -n 2 notes.txt
